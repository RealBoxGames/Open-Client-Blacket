# Open-Client-Blacket

The first open-source client for Blacket.

## Overview

Open-Client-Blacket is not a clone of Blacket. It connects directly to Blacket's official API, letting you sign in with your existing account while providing an expanded feature set beyond what the official client offers.

## Features

- Sign in with your existing Blacket account
- Enhanced usability and quality-of-life improvements
- Tools to streamline gameplay and maximize efficiency
- Fully open source and community-driven
